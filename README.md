# What is this?
    Demo: calling a simple x86_64 ASM function using Rust.

## Tested with
*   Ubuntu 13.04
*   Rust-0.6
*   NASM version 2.10.01

## How can i try this?
    
    $>git clone https://github.com/HOLYCOWBATMAN/rust-asm-demo.git
    $>cd rust-asm-demo
    $>make
